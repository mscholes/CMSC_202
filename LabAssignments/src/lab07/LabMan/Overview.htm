Lab 7: Inheritance

Overview

  1. Description
  2. Source Code Structure
  3. Residence Class
  4. House Class
  5. Tent Class
  6. Apartment Class
  7. Testing

Objectives:

In this lab you will:

  • Learn about inheritance in order to make more specialized derived classes that extend the functionality of a base class.
  • Learn how to inherit instance variables and methods from the base class.
  • Use method overriding to change the behavior of an inherited method when it is used by a more specialized derived class.
  • Learn about the protected visibility modifier.
