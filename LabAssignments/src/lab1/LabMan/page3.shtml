After Eclipse initializes, a screen will appear asking you to designate a workspace. This workspace is a directory structure where Eclipse places your files.

For GL users, designate a directory named CMSC202 under your home directory.

For PC/Laptop users, designate a directory of your choosing.

If this is the first time you've run Eclipse, a screen with several large circles will appear. On the far right, click on the circle labelled “Workbench” and proceed to Step 3. The other circles on the Welcome screen are shortcuts for “Tutorials”, “Overview”, “Samples”, and “What's New”. You can also navigate to the Welcome page via the drop-down menu “Help → Welcome”.
